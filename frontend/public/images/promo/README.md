# Promotional Carousel Images

This folder contains the promotional carousel images for the homepage.

## Required Images

Place the following images in this folder (`frontend/public/images/promo/`):

1. **gaming-setup-1.jpg** - First slide image
   - Recommended size: 1920x400px or similar wide aspect ratio
   - Should feature gaming PC components or setups

2. **gaming-setup-2.jpg** - Second slide image
   - Recommended size: 1920x400px or similar wide aspect ratio
   - Should feature gaming PC components or setups

3. **gaming-setup-3.jpg** - Third slide image
   - Recommended size: 1920x400px or similar wide aspect ratio
   - Should feature gaming PC components or setups

## Image Guidelines

- **Format**: JPG, PNG, or WebP
- **Aspect Ratio**: 16:4 or similar wide format (recommended 1920x400px)
- **File Size**: Keep under 500KB per image for fast loading
- **Content**: Gaming PCs, PC components, gaming setups, or related tech imagery
- **Style**: High-quality, professional images that match the e-commerce theme

## Fallback

If images are not found, the carousel will display a gradient background with emoji icons as placeholders.
